Be Mobile Website Starter

Files included:
- index.html
- products.html
- about.html
- contact.html
- css/styles.css
- js/main.js
- assets/*

How to use:
1. Open index.html in a browser.
2. Replace the sample product content with real products.
3. Add the real store address, opening hours, and social links.
4. Upload the full folder to Netlify, Vercel, GitHub Pages, Hostinger, or cPanel hosting.

Recommended next edits:
- Add real product photos
- Add Sinhala + English text if needed
- Connect the contact form to email or WhatsApp
